# Tank Royale - Project Brief

## Vision
A cross-platform top-down multiplayer tank battle game running on web, iOS, Android, and Xbox. Players compete in dynamic 30x30 grid arenas filled with destructible terrain, power-ups, and AI opponents.

## Core Features
- Cross-platform: WebGL (web), iOS, Android, Xbox
- 30x30 grid arenas with randomly placed destructible crates
- 1 player + 3 AI opponents per match
- Single weapon type with 3 power-ups:
  - Ricochet Bullets (bounces off walls)
  - Armor Bubble (1 extra hit protection)
  - Block Breaker (destroys terrain blocks)
- Single-player challenge mode (10 levels against AI)
- Offline AI mode and online multiplayer (future phase)
- Menu system with key bindings and settings
