# TankRoyale

Top-down Unity tank battler targeting WebGL, iOS, Android, and future Xbox support.

## Current scope (MVP)
- 30x30 grid arena with random destructible blocks
- 1 player + 3 AI tanks
- Base shot + 3 pickups (Ricochet, Armor Bubble, Block Breaker)
- Offline AI mode and 10-challenge single-player mode
- Menu system with key bindings

## Assets
- `toontankslowpoly.unitypackage` (Cartoon Tank Pack)
- `assethunts_gamedev_starter_kit_tanks_v100.unitypackage` (Game Dev Starter Kit)

See `PROJECT_BRIEF.md` and planning docs for details.
